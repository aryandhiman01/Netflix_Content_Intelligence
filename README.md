# 📖 Project Overview

Netflix hosts thousands of Movies and TV Shows from different countries, genres, and audience ratings. Understanding content distribution helps identify production trends, audience preferences, and future content opportunities.

This project performs an end-to-end Exploratory Data Analysis (EDA) on the Netflix Movies & TV Shows dataset using Python.

The project includes:

- 📊 Data Cleaning & Preprocessing
- 📈 Exploratory Data Analysis (EDA)
- 🌍 Country-wise Analysis
- 🎭 Genre Analysis
- ⭐ Rating Analysis
- 📅 Year-wise Content Trends
- 🎬 Movie Duration Analysis
- 💼 Business Insights & Recommendations

---

# 🎯 Problem Statement

Analyze the Netflix dataset to answer questions such as:

- Which content type dominates Netflix?
- Which countries contribute the most content?
- How has Netflix's catalog grown over time?
- Which ratings are most common?
- Which genres are the most popular?
- What business insights can be derived?

# 💼 Top 10 Business Insights

1. Movies account for the majority of Netflix's library.
2. Content production accelerated significantly after 2015.
3. The United States contributes the largest share of titles.
4. International content has grown rapidly.
5. Drama remains the most dominant genre.
6. Mature audience content (TV-MA) is highly prevalent.
7. TV Shows generally have fewer seasons than expected.
8. Recent releases dominate the catalog.
9. Content diversity has increased year by year.
10. Data-driven content acquisition can improve platform growth.
